#projects
Simpl - minimalist notes app

A lightweight note-taking app built with Vue.js. It stores your notes locally using IndexedDB for quick access, and you can export them as TXT files for backup or sharing.

Stream - one flow of news

This app converts RSS feeds into JSON format using the rss2json API. Since it’s hosted on GitHub Pages without server-side support, it relies on this third-party API for functionality.

MovieX - Millions of movies to discover

MovieX is a Vue.js-based app that allows users to explore trending movies. It features a dedicated page where users can view the top popular films. The app utilizes vee-validate for form validation and axios for making API requests to retrieve data from the database.
